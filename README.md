# Cool Burger and Fries - Landing Page Prototype

A visually impressive landing page prototype for Cool Burger and Fries, created as part of Veltrux Agency's prototipo service.

## 🍔 Overview
This prototype showcases the potential website for Cool Burger and Fries, an artisan burger business. Designed to impress clients during presentations, it features:
- Dark food photography aesthetic with vibrant accent colors
- GSAP ScrollTrigger animations for engaging user experience
- Fully responsive design
- WhatsApp Business integration for direct orders
- SEO optimization for local search

## 🛠️ Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion + GSAP ScrollTrigger
- **Deployment**: Cloudflare Pages
- **Language**: TypeScript

## 📱 Features
- Hero section with animated burger illustration
- Menu showcase with 4 featured burgers
- "Why Cool Burger?" differentiators section
- Location information with map placeholder
- WhatsApp Business CTA (floating button and section)
- SEO-optimized metadata and structure
- Responsive design for all device sizes

## 🚀 Deployment
This site is configured for static export and deployment to Cloudflare Pages via GitHub Actions.


## 📁 Project Structure
Following Feature-Sliced Design (FSD) methodology:
```
src/
  app/                 # Next.js routes and layout
  features/            # Domain-specific features (hero, menu, about, etc.)
  shared/              # Reusable components, hooks, utilities
  entities/            # Data models and placeholder data
```
